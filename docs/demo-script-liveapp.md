# Live App 生成演示录屏脚本

## 目标
录制一段 25-35 秒的屏幕录像，展示 Sparo OS 中一句话生成一个可交互 Live App 的完整流程。最终输出为 GIF（推荐 800-1000px 宽，15fps，控制在 5MB 以内放到 README）。

---

## 录前准备

1. 打开 Sparo OS（确保已配置好模型 API Key）
2. 关闭其他窗口，保持桌面干净
3. 打开录屏工具（推荐 **ScreenToGif** 或 **OBS**），框选 Sparo OS 窗口区域
4. 确认输入法是英文状态，避免候选框弹出

---

## 脚本（精确到每一步）

### 第 1 帧：Sparo OS 主界面（0-2 秒）
- 展示 Sparo OS 主窗口，光标停在输入框
- 可选：鼠标悬停在左侧导航栏的 "Code" 应用图标上
- **不做任何操作，给观众 2 秒看懂这是 Sparo OS**

### 第 2 帧：输入提示词（2-8 秒）
- 在输入框中逐字输入：
  ```
  帮我做一个 Markdown 预览器，支持实时渲染和暗色主题
  ```
- 输入速度别太快，让观众能看清你打的是什么
- 按 Enter 发送

### 第 3 帧：Agent 开始工作（8-20 秒）
- Agent 开始思考和执行
- **关键捕捉点**：当聊天流中出现 `InitLiveApp` 工具调用卡片时，这是核心画面
- Agent 会自动创建 Live App 骨架，然后用 Read/Write/Edit 工具填充源码
- 聊天气泡滚动，展示 Agent 的推理和执行过程

### 第 4 帧：Live App 生成完成（20-28 秒）
- Agent 报告完成，提示 "已创建 Markdown 预览器"
- **切换到左侧 Live App 列表/工具箱**，展示新生成的 "Markdown 预览器" 卡片
- 点击打开，展示 Markdown 预览器在独立窗口中运行
- 粘贴一段 Markdown 文本进去，展示实时渲染效果

### 第 5 帧：收尾（28-32 秒）
- 特写：Live App 运行中的画面，展示渲染后的效果
- 淡出或定格

---

## 录屏参数建议

| 参数 | 推荐值 |
|------|--------|
| 工具 | ScreenToGif（免费，Windows） |
| 分辨率 | 800-1000px 宽 |
| 帧率 | 10-15 fps |
| 格式 | GIF |
| 目标大小 | < 5 MB |
| 鼠标 | 显示黄色高亮圆圈 |

---

## 备选文案（如果 Markdown 预览器演示效果不理想）

1. "帮我做一个番茄钟计时器，25分钟专注+5分钟休息"
2. "帮我做一个颜色调色板，支持 HEX 和 RGB 互转"
3. "帮我做一个 JSON 格式化工具"
4. "帮我做一个随机密码生成器"

**选择标准**：生成后 3 秒内就能看到可视化的运行效果。

---

## 录好后放哪里

将生成的 GIF 放在 `png/live-app-demo.gif`，然后在 README 中会被自动引用。
