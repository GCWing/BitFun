# Changelog

All notable changes to taiji-quant will be documented in this file.

## [0.1.0] - 2026-07-23

### Added

- Initial open-source release under MIT License
- 21 taiji crates: trading engine, backtest, realtime, pattern, sentiment, orderflow, strategen, blog-gen, publisher, growth, alert, knowledge-graph, llm, content, bar, abnormal, executor, engine-py, cli, example, strategy-template
- Full BitFun desktop runtime (Rust core + Tauri shell) with Agent, tool contracts, MCP/LSP, remote control
- Web UI + mobile-web frontend
- Desktop CLI + server + relay-server
- Upstream sync workflow (`scripts/sync-upstream.ps1`)
- GitHub open-source governance (Issue/PR/Squash Merge)

### Excluded (closed-source)

- taiji-dvmi: inflection points + dual-line three-state
- taiji-magnet: magnet positioning
- taiji-thrust: three-push detection
- taiji-risk: risk control rules + parameters
